## Getting Started

1. Clone repo

2. Download [Node.js](https://nodejs.org/en/download/)

3. Install all npm packages

```
$ npm install
```

4. Setup .env file

- add .env file at root of project folder

5. Run command to start development server

```
$ npm run start
```
